Djacket

Wheels Re-invented

A free and open source Git server written in Python/Django

Download Djacket

Installation

Download latest release, then extract and install:

tar xzvf djacket-v0.1.0.tar.gz

cd djacket-v0.1.0

sudo bash setup-djacket.sh

For more information about installation and

source code visit https://github.com/Djacket/djacket
